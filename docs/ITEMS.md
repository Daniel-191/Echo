- **Purpose:** Store wealth safely without bank limits

- **Description:** High-value safe investment
- **Cost:** 10,000 credits
- **Purpose:** Store large amounts of wealth safely

- **Description:** Essential tool for digging

- **Cost:** 1,000 credits

- **Note:** Can also be found randomly while using other earning commands

## üî® Craftable Items

Combine materials to create valuable items using `.craft <item>`.

### Joint
- **Ingredients:** 1x Roll + 1x Weed
- **Sell Value:** 10,000 credits
- **Profit:** ~4,000 credits (after material costs)

### Poo
- **Ingredients:** 3x Charcoal + 1x Sulphur
- **Sell Value:** 2,000 credits
- **Profit:** ~600 credits

### C4
- **Ingredients:**
  - 2x Sulphur
  - 1x Charcoal
  - 1x Alarm Clock
  - 5x Potato
  - 2x Electronics
- **Sell Value:** 25,000 credits
- **Special:** Can be used with `.bomb <@user>` command

### Excalibur
- **Ingredients:** 2x Gun + 1x Mythical Sword
- **Sell Value:** 35,000 credits
- **Rarity:** Legendary weapon

### M4A1
- **Ingredients:** 2x Gun + 1x Stick
- **Sell Value:** 30,000 credits
- **Special:** Can be used with `.shoot <@user>` command

### 8 Incher
- **Ingredients:** 1x Stick + 1x David's 4th Ball
- **Sell Value:** 40,000 credits
- **Rarity:** Unique weapon

### Complete Gauntlet
- **Ingredients:**
  - 1x Infinity Gauntlet
  - 1x Legendary Sword
  - 1x David's 4th Ball
- **Sell Value:** 60,000 credits
- **Rarity:** Most powerful item in the game

### Glitch
- **Ingredients:** 2x God
- **Sell Value:** 50,000,000 credits
- **Rarity:** Rarest item in the game
- **Note:** Extremely difficult to craft (God has 0.1% drop rate)

## üí° Economy Tips

### Making Money Fast
1. **Buy a Shovel** - Use `.dig` for best earnings (1,000-1,600 credits/min)
2. **Buy a Bow** - Use `.hunt` as secondary method (600-1,300 credits/min)
3. **Craft Wisely** - Focus on Joint and C4 for best profit margins
4. **Use the Bank** - Deposit money to earn 10% daily interest
5. **Claim Daily** - Never miss your 15,000 credit daily reward

### Item Collection Strategy
1. Keep rare items (< 10% drop rate) for crafting
2. Sell common items (> 30% drop rate) immediately
3. Hoard "God" items if found - needed for Glitch
4. Save Guns, Sticks, and Swords for crafting

### Farming Strategy
1. Plant carrots when you have spare credits
2. 25% profit after 12 hours
3. Max plant 1,000 carrots at once
4. Set a reminder to harvest
